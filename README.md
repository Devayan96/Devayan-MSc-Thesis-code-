# MSc Thesis Code

This repository contains the code used for experiments on differential privacy in binary SVMs against reconstruction attacks.

It is implemented for two datasets:
-Breast Cancer notebook: low-dimensional experiments
-MNIST 3,8 notebook: high-dimensional experiments

The code includes Binary SVM training, Gaussian output perturbation, evaluation of reconstruction attacks, and results on MSE and privacy-utility tradeoff used in the thesis report.

This code is intended as support material for the MSc Thesis report.
